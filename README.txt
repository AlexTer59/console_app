===================Техническая информация об эксплуатации:===================
Перед запуском необходимо:
1. В терминале выполнить команду pip install -r requirements.txt, предварительно создав виртуальное окружение,
если необходимо.
2. В переменных DB_NAME, HOST, USER, PASSWORD, PORT указать данные для вашей базы данных.
